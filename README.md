# BLK Carbon

Website e-commerce untuk brand peralatan olahraga **BLK Carbon**, mencakup produk padel, sepeda, aksesoris, apparel, dan tas.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS** — styling
- **Radix UI** — komponen UI
- **Framer Motion** — animasi
- **Wouter** — routing
- **TanStack Query** — state management

## Halaman

| Route | Halaman |
|---|---|
| `/` | Home |
| `/products` | Semua Produk |
| `/products/:slug` | Detail Produk |
| `/category/:slug` | Kategori |
| `/technology` | Teknologi |
| `/about` | Tentang Kami |
| `/contact` | Kontak |

## Cara Menjalankan

```bash
npm install
npm run dev
```

Build untuk produksi:

```bash
npm run build
```

## Deploy

Project ini di-deploy menggunakan **Vercel**.
